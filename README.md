# 🇦🇷 💻Site sobre a Argentina

Este projeto é um site desenvolvido pelo Projeto Volta ao Mundo, com o objetivo de apresentar informações sobre a Argentina, abordando sua cultura, gastronomia, cidades turísticas e história.

## 📸 Preview
![Preview do site](./assets/img/FaixaArgentina.png)

## 🌎 Venha Conhecer!
O site apresenta conteúdos sobre:<br>
📚- Curiosidades Históricas<br>
💃- Festivais<br>
🧉- Gastronomia típica<br>
🏞️- Cidades turísticas<br>
⚽- Seleção argentina de futebol<br>
